# Information Codes

Visual representation of textual information.

## Quick Response Code

The algorithms are designed to be customizable and generic.

- Version: 1 to 40.
- Data encoding: numeric, alphanumeric, binary or byte, kanji kana.
- Error correction: Reed Solomon L, M, Q, H.
